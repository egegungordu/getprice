# getprice
Python script for showing currency pair prices. 
Scrapes data from [Yahoo Finance](https://finance.yahoo.com/). 


## Dependencies:
bs4  
requests  
argparse  

## Usage:
##### Python3
```
python3 getprice.py <currency_pair>
```
#### optional arguments:
`-a <amount>`
